## 堆栈输入

0. `key`：存储在[storage](/about)中的32字节键值。
1. `value`：要存储的32字节值。

## 示例

| * | Input 1 | * | * | Input 2 |
|--:|--------:|--:|--:|--------:|
| `1` | `0` | * | `1` | `8965` |
| `2` | `0xFFFF` | * | `2` | `0xFF` |

| Storage key before | Storage value |
|-------------------:|--------------:|
| | |

| Storage key after input 1 | Storage value |
|--------------------------:|--------------:|
| `0` | `0xFFFF` |

| Storage key after input 1 then 2 | Storage value |
|---------------------------------:|--------------:|
| `0` | `0xFFFF` |
| `8965` | `0xFF` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
- 当前执行上下文来自 [STATICCALL](/#FA)（自 Byzantium 分叉起）或 [EXTSTATICCALL](/#FB)（自 EOF 分叉起）。
- 当交易剩余 gas 量小于或等于 2300 时（自 Istanbul 分叉起）。
