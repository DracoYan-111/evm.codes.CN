*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`: 要交换的值。
1. 被忽略的值。
2. 被忽略的值。
3. 被忽略的值。
4. 被忽略的值。
5. 被忽略的值。
6. `b`: 要交换的值。

## 堆栈输出

0. `b`: 交换后的值。
1. 被忽略的值。
2. 被忽略的值。
3. 被忽略的值。
4. 被忽略的值。
5. 被忽略的值。
6. `a`: 交换后的值。

## 示例

| * | Input | Output |
|--:|------:|-------:|
| `1` | `1` | `2` |
| `2` | `0` | `0` |
| `3` | `0` | `0` |
| `4` | `0` | `0` |
| `5` | `0` | `0` |
| `6` | `0` | `0` |
| `7` | `2` | `1` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
