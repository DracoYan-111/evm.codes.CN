*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. 被忽略的值。
1. 被忽略的值。
2. 被忽略的值。
3. 被忽略的值。
4. `value`: 要复制的值。

## 堆栈输出

0. `value`: 重复值。
1. 被忽略的值。
2. 被忽略的值。
3. 被忽略的值。
4. 被忽略的值。
5. `value`: 原始值。

## 示例

| * | Input | Output |
|--:|------:|-------:|
| `1` | `0` | `1` |
| `2` | `0` | `0` |
| `3` | `0` | `0` |
| `4` | `0` | `0` |
| `5` | `1` | `0` |
| `6` | | `1` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
- Stack overflow.
