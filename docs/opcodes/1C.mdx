*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 注意事项

将位向最低有效位移动。第一个位之前的位将被丢弃，新移动的位被设置为 0。

## 堆栈输入

0. `shift`：要向右移动的位数。
1. `value`：要移动的 32 字节。

## 堆栈输出

0. `value >> shift`：偏移后的值。如果 `shift` 大于 255，则返回 0。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `1` | `1` | * | `1` | `4` | `0xF` |
| `2` | `2` | | * | `2` | `0xFF` | |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
