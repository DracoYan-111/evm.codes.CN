*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`: 第一个要 加的整数。
1. `b`: 第二个要 加的整数。

## 堆栈输出

0. `a + b`: 加法运算取模结果 2<sup>256</sup>.

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `10` | `20` | * | `1` | `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` | `0` |
| `2` | `10` | | * | `2` | `1` | |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
