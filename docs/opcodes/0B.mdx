*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `b`：要进行符号扩展的整数的大小（以字节为单位，减 1）。
1. `x`：要进行符号扩展的整数值。

## 堆栈输出

0. `y`：符号扩展后的整数结果。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `0` | `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` | * | `1` | `0` | `0x7F` |
| `2` | `0xFF` | | * | `2` | `0x7F` | |

## 错误案例

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
