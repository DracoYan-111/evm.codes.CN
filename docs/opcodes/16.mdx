*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`: first binary value.
1. `b`: second binary value.

## 堆栈输出

0. `a & b`: the 按位与(AND)结果.

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `0xF` | `0xF` | * | `1` | `0xFF` | `0` |
| `2` | `0xF` | | * | `2` | `0` | |

[在 playground 中复现](/playground?unit=Wei&codeType=Mnemonic&code='z1~xF~xwyyz2~~xFw'~yPUSH1%200z%2F%2F%20Example%20y%5CnwFyAND%01wyz~_)。

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
