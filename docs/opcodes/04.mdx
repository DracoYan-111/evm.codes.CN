*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`: 整数 分子
1. `b`: 整数 分母.

## 堆栈输出

0. `a // b`: 整数除法结果. If the 分母 is 0, the result will be 0.

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `10` | `1` | * | `1` | `1` | `0` |
| `2` | `10` | | * | `2` | `2` | |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
