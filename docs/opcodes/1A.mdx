*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `i`: 字节偏移量 从最高有效字节开始。
1. `x`: 32-byte value.

## 堆栈输出

0. `y`：最低有效位置的指定字节。如果字节偏移量超出范围，则结果为 0。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `31` | `0xFF` | * | `1` | `30` | `0xFF` |
| `2` | `0xFF` | | * | `2` | `0xFF00` | |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
