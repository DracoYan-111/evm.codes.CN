*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 注意事项

所有值均视为二进制补码有符号 256 位整数。

## 堆栈输入

0. `a`：左侧有符号整数。
1. `b`：右侧有符号整数。

## 堆栈输出

0. `a > b`：如果左侧较大则为 1，否则为 0。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `0` | `1` | * | `1` | `10` | `0` |
| `2` | `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` | | * | `2` | `10` | |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
