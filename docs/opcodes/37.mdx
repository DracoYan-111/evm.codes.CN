*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 注意事项

对于超出范围的字节，将复制 0。

## 堆栈输入

0. `destOffset`: 结果在[内存](/about)中的字节偏移量，该偏移量将被复制到该位置。
1. `offset`: 要复制的[调用数据](/about)中的字节偏移量。
2. `size`: 要复制的字节大小。

## 示例

| 调用数据 |
|---------:|
| `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` |

| * | Input 1 | * | * | Input 2 |
|--:|--------:|--:|--:|--------:|
| `1` | `0` | * | `1` | `0` |
| `2` | `0` | * | `2` | `31` |
| `3` | `32` | * | `3` | `8` |

| Memory before |
|-------------:|
| `0` |

| Memory after input 1 |
|---------------------:|
| `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` |

| Memory after input 1 then 2 |
|----------------------------:|
| `0xFF00000000000000FFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
