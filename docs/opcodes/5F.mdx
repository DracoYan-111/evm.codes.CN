*索引 1 位于栈顶。*

## 注意事项

新值被放在栈顶，增加所有其他值的索引。因此，特定操作码的值必须按栈的相反顺序推送。例如，对于 [MSTORE](/#52)，第一个推送的值必须是 `value`，然后是 `offset`。

## 堆栈输出

0. `value`: 推送的值，等于 0。

## 示例

| * | * |
|---|---|
| **Code** | `0x5F` |
| **Text** | `PUSH0` |

| * | Input | Output |
|--:|------:|-------:|
| 1 | | `0x00` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- Stack overflow.
