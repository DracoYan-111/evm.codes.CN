*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`: 整数 分子.
1. `b`: 整数 分母.

## 堆栈输出

0. `a % b`: 整数 取模结果。如果 分母 等于 0，则结果为 0。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `10` | `1` | * | `1` | `17` | `2` |
| `2` | `3` | | * | `2` | `5` | |

[在 playground 中复现](/playground?unit=Wei&codeType=Mnemonic&code='z1~3~10wyyz2~5~17w'~yPUSH1%20z%2F%2F%20Example%20y%5CnwyMOD%01wyz~_)。

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
