## 注意事项

成功退出当前[上下文](/about)。

## 堆栈输入

0. `offset`：[内存](/about) 中的字节偏移量（以字节为单位），用于复制此[上下文](/about) 的[返回数据](/about)。
1. `size`：要复制的字节大小（[返回数据](/about) 的大小）。

## 示例

| `1` | Memory |
|--:|------:|
| `1` | `0xFF01` |

| * | Input |
|--:|-----:|
| `1` | `0` |
| `1` | `2` |

| `1` | Calling context return data |
|--:|---------------------------:|
| `1` | `0xFF01` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
