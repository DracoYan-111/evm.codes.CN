*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 注意事项

停止当前上下文执行，恢复状态更改（有关状态更改操作码的列表，请参阅 [STATICCALL](/#FA)）并将未使用的气体返回给调用者。
它还将天然气退款恢复到当前上下文之前的值。
如果使用 REVERT 停止执行，则值 0 将被放入调用上下文的堆栈中，调用上下文将继续正常执行。
调用上下文的[返回数据](/about)被设置为该上下文的给定内存块。

## 堆栈输入

0. `offset`：[内存](/about) 中的字节偏移量（以字节为单位）。调用上下文的返回数据。
1. `size`：要复制的字节大小（[返回数据](/about) 的大小）。

## 示例

| `1` | Memory |
|--:|------:|
| `1` | `0xFF01` |

| * | Input |
|--:|-----:|
| `1` | `0` |
| `1` | `2` |

| `1` | Calling context return data |
|--:|---------------------------:|
| `1` | `0xFF01` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
