*索引 1 位于栈顶。*

## 堆栈输入

0. `key`: 32 字节密钥位于[临时存储](/about)。
1. `value`: 32 字节值要存储。

## 示例

| * | Input 1 | * | * | Input 2 |
|--:|--------:|--:|--:|--------:|
| `1` | `0` | * | `1` | `8965` |
| `2` | `0xFFFF` | * | `2` | `0xFF` |

| 临时存储密钥 | 临时存储值 |
|-----------:|--------------:|
| | |

| 临时存储密钥 | 临时存储值 |
|-----------:|--------------:|
| `0` | `0xFFFF` |

| 输入 1 和 2 后，瞬态存储密钥 | 临时存储值 |
|---------------------------------:|--------------:|
| `0` | `0xFFFF` |
| `8965` | `0xFF` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- Stack overflow.
