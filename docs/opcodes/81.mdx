*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`: 被忽略的值。
1. `b`: 要复制的值。

## 堆栈输出

0. `b`: 重复值。
1. `a`: 被忽略的值。
2. `b`: 原始值。

## 示例

| * | Input | Output |
|--:|------:|-------:|
| `1` | `0` | `1` |
| `2` | `1` | `0` |
| `3` | | `1` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
- Stack overflow.
