## 注意事项

对于超出范围的字节，将复制 0。

## 堆栈输入

0. `mem_offset`: 结果将复制到的[内存](/about)中的字节偏移量.
1. `offset`: 要复制的[数据段](/about)中的字节偏移量.
2. `size`: 要复制的字节大小.

## 示例

| Data Section |
|---------|
| `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` |

| Memory before |
|---------|
| `0x000020` |

| Memory after |
|---------------------:|
| `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 在遗留（非 EOF）上下文中执行。
