*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`：整数基数。
1.`指数`：整数指数。

## 堆栈输出

0. `a ** exponent`: 指数运算模 2<sup>256</sup> 的整数结果。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `10` | `100` | * | `1` | `2` | `4` |
| `2` | `2` | | * | `2` | `2` | |

[在 playground 中复现](/playground?unit=Wei&codeType=Mnemonic&code='z1~2~10wyyz2~2~2w'~yPUSH1%20z%2F%2F%20Example%20y%5CnwyEXP%01wyz~_)。

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
