## 堆栈输出

0. `size`: [calldata](/about) 的字节大小。 

## 示例

| Calldata |
|---------|
| `0xFF` |

| * | Input | Output |
|--:|------:|-------:|
| `1` | | `1` |

[在 playground 中复现](/playground?unit=Wei&callData=0xFF&codeType=Mnemonic&code='CALLDATASIZE'_)。

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- Stack overflow.
