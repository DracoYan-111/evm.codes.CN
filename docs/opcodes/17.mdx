*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `a`：第一个二进制值。
1. `b`：第二个二进制值。

## 堆栈输出

0. `a | b`: 按位或运算的结果。

## 示例

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `0xF0` | `0xFF` | * | `1` | `0xFF` | `0xFF` |
| `2` | `0xF` | | * | `2` | `0xFF` | |

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
