*索引 1 位于栈顶。参见 [PUSH](/#60)。*

## 堆栈输入

0. `value`: 要复制的值。

## 堆栈输出

0. `value`: 重复值。
1. `value`: 原始值。
## 示例

| * | Input | Output |
|--:|------:|-------:|
| `1` | `1` | `1` |
| `2` | | `1` |

[在 playground 中复现](/playground?unit=Wei&codeType=Mnemonic&code='zSet%20styPUSH1%201~~zDuplicyDUP1'~%5Cnz%2F%2F%20yate~%01yz~_)。

## 错误情况

在以下情况下，当前上下文所做的状态更改会被[reverted](#FD)：
- Gas 不足。
- 堆栈上的值不足。
- Stack overflow.
