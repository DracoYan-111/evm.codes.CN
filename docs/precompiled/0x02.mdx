## Inputs

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| `[0; length[` | data | 使用 SHA2-256 进行哈希处理的数据 |

## Output

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| [0; 31] (32 bytes) | hash | 结果哈希 |

如果没有提供足够的gas，则没有返回数据，表明预编译合约错误。
