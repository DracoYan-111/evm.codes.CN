## 注释

恒等函数通常用于复制一块内存。
## Inputs

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| `[0; length[` | data | 要返回的数据 |

## Output

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| `[0; length[` | data | 输入数据 |

如果没有提供足够的gas，则没有返回数据，表明预编译合约错误。
## 例子

| Input | Output |
|------:|-------:|
| `0xFF` | `0xFF` |
