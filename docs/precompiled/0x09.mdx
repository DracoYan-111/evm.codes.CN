## Inputs

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| `[0; 3]` (4 bytes) | rounds | 轮数（大端无符号整数） |
| `[4; 67]` (64 bytes) | h | 状态向量（8 个 8 字节小端无符号整数） |
| `[68; 195]` (128 bytes) | m | 消息块向量（16个8字节小端无符号整数） |
| `[196; 211]` (16 bytes) | t | 偏移计数器（2 个 8 字节小端整数） |
| `[212; 212]` (1 bytes) | f | 最终块指示器标志（0或1） |

## Output

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| `[0; 63]` (64 bytes) | h | 状态向量（8 个 8 字节小端无符号整数） |
