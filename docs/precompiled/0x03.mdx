## Inputs

| 字节范围| Name | 描述 |
|-----------:|-----:|------------:|
| `[0; length[` | data | 使用 RIPEMD-160 进行哈希处理的数据 |

## Output

| 字节范围 | Name | 描述 |
|-----------:|-----:|------------:|
| [0; 31] (32 bytes) | hash | 结果20字节散列右对齐到32字节 |

如果没有提供足够的gas，则没有返回数据，表明预编译合约错误。

| Input | Output |
|------:|-------:|
| `0xFF` | `2c0c45d3ecab80fe060e5f1d7057cd2f8de5e557` |
